Faster dock appearing

November 30, 2014

In macOS if we set the Dock to automatically hide and show we’ll notice a slight delay between reaching the bottom of the screen and the Dock appearing.

For it to show immediately open a terminal and run these three lines:

defaults write com.apple.dock autohide-delay -int 0
defaults write com.apple.dock autohide-time-modifier -float 0.4
killall Dock

To revert the changes:

defaults delete com.apple.dock autohide-delay
defaults delete com.apple.dock autohide-time-modifier
killall Dock
