Better Django shell

September 22, 2015

Using django-extensions and bpython we’ll add autocompletion, syntax highlighting and automatic import of our models to the Django shell.

Install these packages:

pip install django-extensions
pip install bpython

Now we add django-extensions to our Django project applications. Edit the app/app/settings.py file:

INSTALLED_APPS = (
    ...
    'django_extensions'
)

Now we can launch the improved shell:

python manage.py shell_plus
