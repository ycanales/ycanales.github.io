Excluding paths in Sublime Text search

July 20, 2016

When using the advanced search in Sublime Text—⌘+⇧+F in macOS or Ctrl+⇧+F in Windows—we can exclude files and directories from using the Where field, for instance:

-*project/dist*

For several exclusions:

-*project/dist*, -*project/node_modules*
