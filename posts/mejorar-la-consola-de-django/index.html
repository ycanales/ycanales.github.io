Mejorar la consola de Django

September 22, 2015

Con django-extensions y bpython le agregaremos a la consola de Django import automático de nuestros modelos, syntax highlight y autocompletado del código.

Instalamos los paquetes:

pip install django-extensions
pip install bpython

Ahora en nuestro proyecto Django agregamos django-extensions a las aplicaciones. Editamos el archivo app/app/settings.py:

INSTALLED_APPS = (
    ...
    'django_extensions'
)

Ahora podemos lanzar la consola mejorada:

python manage.py shell_plus
